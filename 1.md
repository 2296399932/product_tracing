商品追溯系统技术文档
1. 系统概述
1.1 系统目标
构建一个完整的商品追溯系统，实现商品从生产到销售的全程追踪，确保商品质量和安全性，提供透明的商品信息查询服务。
1.2 技术架构
后端：Django + Django REST framework
前端：Vue.js + Element UI
数据库：MySQL
缓存：Redis
消息队列：Celery
认证：JWT
